# Tool Refund v1.0.0

Tool Refund is a Silksong mod that restores spent tools when you die.

## Installation

To install this mod, place the `ToolRefund` folder (containing `ToolRefund.dll`) in your
`BepInEx/plugins` folder.

## Usage

Dying after using tools will restore all tools used since you last rested or autosaved.

This mod can be configured by editing `Esper89.ToolRefund.cfg` in your `BepInEx/config` folder,
or (if you have Configuration Manager installed) in-game by pressing the **F1** key.

## Repository

This mod's source code can be found in its
[GitHub repository](https://github.com/Esper89/Silksong-ToolRefund).
