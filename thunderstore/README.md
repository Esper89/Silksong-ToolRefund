# Tool Refund

Tool Refund restores spent tools when you die.

## Installation

This mod requires
[BepInEx 5](https://thunderstore.io/c/hollow-knight-silksong/p/BepInEx/BepInExPack_Silksong), and is
installed to your `BepInEx/plugins` directory.
[Configuration
Manager](https://thunderstore.io/c/hollow-knight-silksong/p/Yukikaco/BepinExConfigurationManager)
can be used with this mod.

## Usage

Once this mod is installed, by default, dying after using tools will restore all tools used since
you last rested or autosaved.

This mod can be disabled by editing its config file at `BepInEx/config/Esper89.ToolRefund.cfg` or
using Configuration Manager to set `Enabled = false` in the `[General]` section.

## Repository

This mod's source code can be found in its
